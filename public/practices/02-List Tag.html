List Tag

ภาษา HTML จะมี TAG สำหรับแสดงผลที่เป็นรายการ(list) อยู่ 3 ประเภทคือ

  1. รายการแบบใช้สัญลักษณ์
  2. รายการแบบใช้ตัวเลข
  3. ราบการแบบหัวข้อและเนื้อหา

การรายงานแบบใช้สัญลักษณ์

จะใช้TAGชื่อ ul เป็น TAG หลัก และเราจะใส่TAG ที่มีชื่อว่า li ข้างหน้า TAG ul อีกที
โดย TAG li 1 TAG จะใช้สำหรับ 1 รายการ ถ้ามี 3 รายการก็หมายความว่าจะมี TAG li อยู่ใน TAG ul 3 TAG

การรายงานแบบใช้ตัวเลข

จะมีรูปแบบคล้ายกับรายการแบบสัญลักษณ์โดยเราจะเปลี่ยนTAG หลังจาก ul เป็น ol ก็จะมี TAG li เป็นตัวกำหนดรายการตามเดิม

ตัวอย่างเช่น

เหตุผลหลักที่ทำให้การเรียนเขียนโปรแกรมเป็นไปอย่างยากลำบาก

  1. อ่านภาษาอังกฤษไม่ออก
  2. จำคำศัพท์ไม่ได้
  3. ไม่จดคำศัพท์เฉพาะ

รายการแบบหัข้อและเนื้อหา

รูปแบบนี้จะแตกต่างจากสองแบบแรกอยู่พอสมควรคือ TAG หลักคือ dlแต่ภายในจะไม่ใช้ TAG li
โดยจะปลี่ยนไปใช้ 2 TAG ด้วยกันคือ dt ใช้สำหรับรายการที่เป็นหัวข้อ และ dd ใช้สำหรับรายการที่เป็นเนื้อหา
และแน่นอนว่า ทั้ง 2 TAG นี้จะต้องอยู่ใน TAG dl
